# 元素类型

## 图片对象（Category.Image）

![](https://cdn.nlark.com/yuque/0/2026/png/42539759/1767497368467-bf2d5a67-d784-4d87-b3e6-d4cb6ca9e609.png)

图片对象。交互状态随编辑器模式变化，渲染层级位于底层。

## 绘制矩形（Category.DrawRect）

![](https://cdn.nlark.com/yuque/0/2026/png/42539759/1767497409225-dbc779ae-345f-4c2e-bc39-e4943132c2c9.png)

绘制矩形。交互状态随编辑器模式变化，渲染层级位于底层。

## 区域标注框（Category.Region）

![](https://cdn.nlark.com/yuque/0/2026/png/42539759/1767497422383-a6777d31-b06b-4c3b-a69b-9c8c85603c39.png)

区域标注框，附着于 `Category.Image` 或 `Category.DrawRect` 对象上。不响应用户交互，渲染层级始终置顶。

## 标记点（Category.Marker）

![](https://cdn.nlark.com/yuque/0/2026/png/42539759/1767497441888-c3513d27-700d-429e-91ba-a2c1b2fbd752.png)

标记点，附着于 `Category.Image` 或 `Category.DrawRect` 对象上。仅在 Select 模式下响应交互，渲染层级始终置顶（高于 Region）。
