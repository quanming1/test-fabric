# 编辑器模式

编辑器通过 `ModePlugin` 管理模式状态。

## 选择模式（EditorMode.Select）

可选择、框选、移动对象。

- 画布：支持框选，鼠标为默认箭头
- Image / DrawRect：可选中、可交互
- Marker：可交互
- Region：不可交互

## 拖拽模式（EditorMode.Pan）

拖拽画布进行平移。

- 画布：禁用框选，鼠标为抓手
- Image / DrawRect：不可选中、不可交互
- Marker：不可交互
- Region：不可交互

## 绘制矩形模式（EditorMode.DrawRect）

在画布上绘制矩形对象。

- 画布：禁用框选，鼠标为十字准星
- Image / DrawRect：不可选中、不可交互
- Marker：不可交互
- Region：不可交互

## 区域标注模式（EditorMode.RangeSelect）

在目标对象上绘制区域标注框或添加标记点。

- 画布：禁用框选，鼠标为十字准星
- Image / DrawRect：不可选中、可交互（用于检测鼠标事件）
- Marker：不可交互
- Region：不可交互

> 按住 Ctrl/Meta 并悬浮在可标记对象上时，自动切换至 RangeSelect 模式。
